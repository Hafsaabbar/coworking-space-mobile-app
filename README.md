# EspaceCoworking 

A mobile application for coworking space reservation and management developed as part of a mobile development academic project at ENSIAS.

## Overview

**EspaceCoworking** is an Android mobile application designed to simplify the management and reservation of coworking spaces.  
The platform connects clients looking for flexible workspaces with owners who want to efficiently manage their coworking spaces and reservations.

The application provides:
- Real-time coworking space availability
- Reservation management
- User authentication
- Notifications system
- Calendar integration
- Dedicated owner dashboard

---

## Features

### Client Features
- User registration & authentication
- Browse available coworking spaces
- Search & filter spaces
- View workspace details
- Make reservations
- Cancel reservations
- Track reservation history
- Receive confirmation notifications
- Calendar integration
- Manage personal profile

### Owner Features
- Add coworking spaces
- Update and delete spaces
- Manage reservations
- Accept or reject booking requests
- Monitor workspace availability
- Owner dashboard

## System Architecture

The application is structured around two main modules:

### Client Module
Allows users to:
- Search coworking spaces
- Reserve workspaces
- Track reservations
- Manage profiles

### Owner Module
Allows owners to:
- Manage spaces
- Handle reservation requests
- Monitor workspace activity
